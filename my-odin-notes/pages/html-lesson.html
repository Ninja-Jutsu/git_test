Basic HTML(hypertext markup language) Structure

    Every HTML file will have to start at least with the following elements:

  • !DOCTYPE html,telling the browser or the code editor that it is an HTML file.
  • Html, selecting your coding language.
  • Head, containing all the extra elements such as the author's name, the version of html used, as well as the title appearing on top.
  • The Body, which will contain everything else our page will be displaying.

The Body

    The body is the main part of our html file. It is in this part that you can do your magic. All of it. It should contain your headings, your paragraphs and whatever other elements you are intending to include The body contains your:

  • Paragraphs: this is the simplest element of an html. It simple separates text. It makes paragraphs
  • Headings: h1, h2 and h3 and so on until h6. The bigger the number the smaller is the font size
  • Div: A div is a container. it puts the content inside a box you can manipulate
  • Span: A span selects a text inside any other elements ( p or div ... ) in order to apply some settings on it
  • emphasize: we can use "em" to put text in italic
  • strong: we can use "strong" to put text in bold
