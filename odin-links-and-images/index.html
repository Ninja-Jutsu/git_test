the building of an HTML file

Every HTML file will have to start with the following elements:

  • !DOCTYPE html,telling the browser or the code editor that it is an HTML file.
  • Html, selecting your coding language.
  • Head, containing all the extra elements such as the autor's name, the version of html used, as well as the title appearing on top.
  • body, which will contain everything else our page will be displaying.
