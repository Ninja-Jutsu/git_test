the body

The body is the main part of our html file. It is in this part that you can do your magic. All of it.

It should contain your headings, your paragraphs and whatever other elements you are intending to include

The body contains your:

  • Paragraphs this is the simplest element of an html. It simple seperates text. It makes paragraphs
  • Headings: h1, h2 and h3 and so on until h6. The bigger the number the smaller is the font size
  • Div: A div is something similar to a paragraph. it puts the content inside a box you can manipulate
  • Span: A span selects a text inside any other elemets ( p or div ... ) in order to apply some settings on it
  • emphasize: we can use "em" to put text in italic
  • strong: we can use "em" to put text in bold
  • img: The only way to include an image so far. must include " src="https://fr.web.img6.acsta.net/r_654_368/newsv7/21/02/22/16/03/4655709.jpg" alt="the newset batman sign"
