Part 3:Links

How can I use links?

Let's use the following line as an example:

href="https://www.theodinproject.com/about" target="_blank" rel="noopener noreferrer">click me

  • Anchor: we use "a" attribute to attach links to our files
  • href: specifies the destination of the link. It could be a relative or a Absolute link
  • target: by default the browser would open the link on the same tab. Use target + _blank to open it on a new tab
  • rel: it describes the relation between the current page and the linked document.

    We can add two very important options to the rel attribute:

    "noopener" it prevents the owner of the link to get access to your link.

    "noreferrer" if you wish to not let the opened link know that your webpage links to it

CSS the link

Notes of the most important options to add to your css for links:

  • visiteed:It will turn the link to a specific color when opened already
  • hover:Will change the color of the link when moving on top of it using the mouse
  • active: for a fraction of a second when you click on the link it changes appearances
  • the order of these 3 options is very important.
